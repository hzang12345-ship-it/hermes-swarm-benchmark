## Build Sequence (What Worked)

1. `benchmark_quick.py --run-via-agent` → writes `/tmp/omega_goal.txt`
2. Agent (with skill loaded) reads goal → calls `delegate_task(role='orchestrator')`
3. OMEGA spawns sub-agents, collects results, generates PIL chart, returns summary

## Bugs Caught During Build

| Bug | Symptom | Fix |
|-----|---------|-----|
| `.format()` on f-string brace | `KeyError: 'time'` at script construction | Use `%%` + `.replace("NAME","{name}")` |
| compute_pi inline script truncated | Sub-agents ran 100k not 5M iterations | OMEGA goal too large; use temp-file approach |
| `run_test()` always returned `simulated` | No real agent execution | Rewrite to call agent via `delegate_task` |
| `hermes acp` designed for interactive stdio | Not batch-friendly | ACP JSON-RPC reserves stdout; logging → stderr |
